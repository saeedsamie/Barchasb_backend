
# Barchasb Backend

This is the backend service for the Barchasb crowdsourcing application.

## How to Run

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Start the server:
   ```bash
   uvicorn app.main:app --reload
   ```

3. Access the API docs at `http://127.0.0.1:8000/docs`.
